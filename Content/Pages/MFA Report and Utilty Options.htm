Data Export/Import

There are 2 new Utility options related to User Profile that can assist with updating User Profile records before implementing Multi-Factor Authentication. Please see the Before You Begin help file for details on using these options for updating existing records for MFA implementation.

 

Data Export > User Profile Export

This option can be used to export existing User Profile data which also includes linked Physician and Employee IDs, Names, Mobile Phones and Email addresses and can be used to prepare for implementing MFA.

 

Once the existing data is exported, the User Profile Mobile Phone and Email address can be updated by copying the Physician/Employee columns to the User columns, then the export can be used in the User Profile (Import or Update) option to update the User Profiles.

 

Please see Utility > Data Export for instructions on generating a file that can be updated.

 

Data Import > User Profile (Import or Update)

This new option provides a way for a center to update their User Profiles with the most current Mobile Phone and Email from the Physician/Employee data in bulk before going live with MFA.

Once the User Profile data has been manipulated in the export file, its ready for Validation and Importing.

 

Please see Utility > Data Import for details on validation and importing files..

 

PLEASE NOTE:

New users can also be imported through this option. They can be added to a worksheet being used for updating existing users, or a new import worksheet can be created for just new Users.
